
## Stack

- TypeScript
- NextJS 15
- React 19
- Tailwind 4
- React Redux 9
- Redux Toolkit 2


## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
